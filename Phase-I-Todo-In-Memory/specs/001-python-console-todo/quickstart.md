# Quickstart: Phase I: Todo In-Memory Python Console App

This guide explains how to run the console application.

## Prerequisites

- Python 3.11+
- `uv` package manager (or `pip`)

## Running the Application

1.  **Navigate to the project root directory.**

2.  **Run the application as a module:**

    ```bash
    python -m todo_console_app.main
    ```

3.  **Interact with the application:**

    The application will display a menu of options. Enter the number corresponding to your desired action and follow the prompts.

    ```
    --- To-Do List Menu ---
    1. Add Task
    2. View Tasks
    3. Mark Task as Complete
    4. Update Task
    5. Delete Task
    6. Quit
    -----------------------
    Enter your choice:
    ```
